### RazorPagesPizza


Simple Pizza internal inventory management implement CRUD operation such as Adding new Pizza, Deleting Pizza form Inventory , Update Pizza, and more.       
## Tools: 
- ASP.NET WebApp 
- Razor Pages 
- .NET CLI 
- .NET 6.0
